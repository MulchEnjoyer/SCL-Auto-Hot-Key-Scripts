# OuraPasswordResetAHK
A macro intended to be used to reset passwords based off a spreadsheet.
Below is the config adjust as required, note that the pixel detection was created with the following settings, use AHK window spy to reconfigure.
Software: Chrome on Windows 10 100% Zoom
Hardware: 27inch 2k screen
Spreadsheet: Columns must be in the following order: URL, Password, Password Reset Status. Status must be set up on a minimumm to be conditional to accept Y to change cell colour to GreenColor := 0xB7E1CD.
